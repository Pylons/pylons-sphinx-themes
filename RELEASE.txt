Releasing
=========

`zest.releaser <http://zestreleaser.readthedocs.io/en/latest/>`_ is used for releasing this package.

To release, invoke the following command from your virtual environment.

.. code-block:: bash

    fullrelease


Updating README
---------------

README.rst is the canonical README file in reStructuredText format for use on PyPI. Edit this file to update both READMEs.

README.md is the markdown version. It is generated through the use of pandoc using the following command.

.. code-block:: bash

    pandoc --from=rst --to=markdown --output=README.md README.rst
